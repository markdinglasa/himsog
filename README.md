## TO START

- install requirements
- restore database backup (if applicable)

- Backend/Server

1. npm install
2. setup .env
3. npm run serve or npm run start <!-- server/backend side >

- Frontend/Client

1. npm install
2. npm run dev <!-- client/frontend side >
3. open a brower
